SakugaGP is a collection of Free Grease Pencil addons made by 2D Animator for 2D Animators.

Download : Gumroad Github

Requirement : Blender v5.x

Download the addons via Gumroad to get email update notifications, also you can ask questions or report any bug on SakugaGP's Discord Server.

Installation

*Better uninstall the old addons first before installing the new versions.

Read blender docs : How to Install Addons
You can also Drag & Drop the Addon .zip file one-by-one from your File Manager into Blender to install them.

Tips

  • You can Right Click on any button > Assign Shortcut.. to Assign/Change/Remove Shortcut to that button.

Exporter

  • Turn On/Off Antialiasing

  • Quicky set render output as Video/Image with or without transparency 

  • Render Keyframes : This feature is for rendering the keyframes (cels) only 

  • You have to set render to image formats (JPG/PNG/TGA) before rendering the keyframes. 

  • You can change the filename for exported images as Counting Numbers or Frame Numbers. 

  • By default the keyframes data is referenced to layers that named "data" in every Grease Pencil Objects that you put in the list. 

  • You can export the keyframes data as .xdts with Export XDTS button, also Copy Command is for copying the keyframe data as text command , this command is meant to be pasted into Auto-Sheet (a Timesheet web-tool by Moaang https://moaang.github.io/auto-sheet/ ) 

  • Rendered Keyframes are exported inside your project file's current folder inside "export" folder. the Exported .xdts file as well. 

  • Render Current Layer is for rendering the keyframe of active layer of the active Grease Pencil Object, the render output path for it is based on the Blender's output path in the Output Properties 

Extra Tools

User customizable buttons in the toolbar
  • For first time use, try click Tools > Default Buttons Set & Save JSON 

  • Click Open JSON Path to find the ExtraTools.json & ExtraTools folder , you can edit the json file to rearrange or add new buttons . inside the ExtraTools folder there are the scripts for those buttons. 

  • Click Tools > Load JSON to reload the extra tools's buttons list in the toolbar 

Brush Sizes

Brush Sizes Palette like Krita/CSP
  • For first time use, similar to Extra Tools addon, you need to create the json file first , go to N-Panel > SakugaGP > Brush Sizes panel) click Brush Sizes > New JSON & Load JSON

  • Click Open JSON Path to find the BrushSizes.json , inside the JSON contains list of brush sizes that you can add/remove/edit 

  • After edit & saving the JSON file, click Brush Sizes > Load JSON to reload it 

Camera

Adds Camera Mirror/Rotation without leaving Grease Pencil Draw Mode and Import Animation Template
  • Inside the "Camera popover menu" there are button for Importing Animation Template , Switch Camera/Overscan , and below it are Overscan properties

  • Import Template : Importing Template will create a Mesh Plane Image into 3D Viewport ,then it'll auto update the overscan value same as the image's resolution. it'll also automatically create a separate Action for normal Camera(Camera focus on animation) and Overscan (Template size) named "CameraOverscan" action.

  • Switch Camera/Overscan : You can Switch Between Original Camera resolution with Camera Overscan resolution (Template size) wile also switching between camera actions named "CameraAction" and "CameraOverscan"

  • Tutorial How to import a Anime Template and Switch between Camera size / Overscan (Template size) :

  • In the panel, there are a toggle and slider for controlling Camera's Passepartout Alpha (The dark area outside canvas) 

  • Below it is a Reset Camera Rotation button (to zero) and a slider to rotate the Camera

  • Below the rotation reset button is a Arrow button for selecting the Active Camera used in the scene 

  • Next are X and Y buttons for mirroring the Camera Horizontal/Vertically

  • The L and R button are for rotating the Camera Counter-clockwise/Clockwise

More about the Overscan feature :

Overscan is a feature that makes Camera resizing behave like canvas resizing similar to other drawing programs (the Overscan script is originated from https://projects.blender.org/pioverfour/camera_overscan)

Manager

  • Object Menu : Add Grease Pencil Object with or without all Materials

  • Material Menu : Add New Stroke/Fill Materials with one click 

  • Material Menu : Link All Materials into Active Grease Pencil Object's material slots

  • Material Menu : Unlink All Materials that presents in the Active Grease Pencil Object's material slots 

  • Arrange the positions of Grease Pencil Object to Front/Back (Y axis)  

  • Opacity toggle to quickly add the opacity modifier to Active Greace Pencil Object, also works for imported Mesh Plane and Reference Image Objects 

  • Freeze Frame (For Shift & Trace/Tap Wari) Click the colored buttons to automatically duplicate and adding Time Offset and Opacity modifier to Active Grease Pencil Object. the Time Offset Modifier is automatically set to "Fixed Frame" to current frame . (tips : you can reset Grease Pencil Objecct's Transformation with shortcuts Alt+R/Alt+G/Alt+S) 

Extras

Extra features for SakugaGP
  • Prev/Next buttons for Keyframe Jump Cycle, for example if the current keyframe is the last key, it'll jump back to the first key. this will be useful for when making looping animations. You can skip other type of Keyframes as well 

  • Timer , when countdown is over it can play Image Sequence over the 3D Viewport and a Audio file, click the gear icon to see the preferences for change the Image Sequence/Audio , Frame Duration , Timer Duration

Stamps

  • Insert Text directly into Grease Pencil or as Text Object(The Textbox supports Multi-line, You can press Shift+Enter to create a new line in the textbox)

  • And, Draw the Outline of Camera Into Grease Pencil (Make sure you select a Grease Pencil Object First)
