SakugaGP is a collection of Free Grease Pencil addons made by 2D Animator for 2D Animators.

Download : Gumroad Github

Requirement : Blender v5.x

Download the addons via Gumroad to get email update notifications, also you can ask questions or report any bug on SakugaGP's Discord Server.

Exporter

  • Turn On/Off Antialiasing

  • Quicky set render output as Video/Image with or without transparency 

  • Render Keyframes : This feature is for rendering the keyframes (cels) only 

  • You have to set render to image formats (JPG/PNG/TGA) before rendering the keyframes. 

  • You can change the filename for exported images as Counting Numbers or Frame Numbers. 

  • By default the keyframes data is referenced to layers that named "data" in every Grease Pencil Objects that you put in the list. 

  • You can export the keyframes data as .xdts with Export XDTS button, also Copy Command is for copying the keyframe data as text command , this command is meant to be pasted into Auto-Sheet (a Timesheet web-tool by Moaang https://moaang.github.io/auto-sheet/ ) 

  • Rendered Keyframes are exported inside your project file's current folder inside "export" folder. the Exported .xdts file as well. 

  • Render Current Layer is for rendering the keyframe of active layer of the active Grease Pencil Object, the render output path for it is based on the Blender's output path in the Output Properties 

Extra Tools

User customizable buttons in the toolbar
  • For first time use, try click Tools > Default Buttons Set & Save JSON 

  • Click Open JSON Path to find the ExtraTools.json & ExtraTools folder , you can edit the json file to rearrange or add new buttons . inside the ExtraTools folder there are the scripts for those buttons. 

  • Click Tools > Load JSON to reload the extra tools's buttons list in the toolbar 

Brush Sizes

Brush Sizes Palette like Krita/CSP
  • For first time use, similar to Extra Tools addon, you need to create the json file first , go to N-Panel > SakugaGP > Brush Sizes panel) click Brush Sizes > New JSON & Load JSON , 

  • Click Open JSON Path to find the BrushSizes.json , inside the JSON contains list of brush sizes that you can add/remove/edit 

  • After edit & saving the JSON file, click Brush Sizes > Load JSON to reload it 

Camera Extras

Adds Camera Mirror/Rotation without leaving Grease Pencil Draw Mode
  • Select Camera is for selecting the Active Camera used in the scene 

  • Below it is a toggle and slider to control the Camera's Passepartout Alpha (The dark area outside canvas) 

  • Below it is a Reset Camera Rotation button (to zero) and a slider to rotate the camera, you can also rotate the camera by checking the Rotation Shortcuts checkbox to enable the ALT+Comma / ALT+Dot keyboard shortcuts to rotate the camera. 

  • The X and Y button are for mirroring the camera 

+ Overscan

Camera resize behave like canvas resizing
  • Switch Camera Action button is for switching the Active Camera's action, between actions named "CameraAction" and "CameraOverscan" while also toggling the "Overscan" checkbox. 

  • New X/Y are your camera new size, its used for resizing the camera without removing the original camera size. 

  • "CameraAction" -> uses original camera size, while "CameraOverscan" uses the new camera size 

  • This Camera Extra's Overscan feature is useful for those using Anime Template to switch between Camera focusing on the animation with 16:9 size for example, and the full image cel/template size for exporting the image cels. 

Grease Pencil Manager

  • Arrange the positions of Grease Pencil Object to Front/Back (Y axiis)  

  • Opacity button to quickly add the opacity modifier to Active Greace Pencil Object, also works for imported Mesh Plane and Reference Image Objects 

  • Freeze Frame (For Shift & Trace/Tap Wari) Click the colored buttons to automatically duplicate and adding Time Offset and Opacity modifier to Active Grease Pencil Object. the Time Offset Modifier is automatically set to "Fixed Frame" to current frame . (tips : you can reset Grease Pencil Objecct's Transformation with shortcuts Alt+R/Alt+G/Alt+S)
